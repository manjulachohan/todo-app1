# Todo App

A simple in-memory CLI Todo application built using spec-driven development.

## Features

- Add new tasks
- View all tasks
- Update existing tasks
- Delete tasks
- Mark tasks as complete/incomplete

## How to Run

1. Make sure you have Python installed
2. Run the application: `python src/main.py`
3. Follow the on-screen prompts

## Project Structure

- `constitution/` - Project constitution and principles
- `specs/` - Current specifications
- `history/` - Historical specification versions
- `src/` - Source code
